To prove their identity via the Patient Online (POL) journey, a user must use NHS login to enter the POL Account Linkage Key/Passphrase, Account ID and Practice ODS code which will have been issued to the user when they first requested GP online service credentials from their GP surgery.

Once this information is submitted by the user, the details are checked against the GP system. As in the PYI journey, during this process the NHS number will also be matched to the user and stored in their NHS login account.
