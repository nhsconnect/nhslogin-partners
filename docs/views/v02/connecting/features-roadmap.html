Features and Roadmap

NHS login roadmap

Please note: As the service develops we are defining our roadmap based on research and data insights. The below features are planned, but might move around as we respond through our beta phase.

JANUARY - MARCH 2019

  • Enable users to amend the phone number and email address linked to their account
  • Users will be prompted to consent to share data with services
  • Passwordless login for apps and devices that support biometric unlocking Automation of ID checks
  • Support for worldwide government issued ID documents

APRIL - JUNE 2019

  • Seemless switching between services using NHS login (single sign-on)
  • Delegated / Proxy access (limited trial)
  • Ability for users to switch between desktop and mobile devices to assist ID checks
  • Offline vouching and user invite to prescribed apps

JULY ONWARDS

  • Passwordless login for browser based services
  • Offline ID checks at locations other than GP practices
